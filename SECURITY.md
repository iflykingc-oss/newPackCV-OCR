# 安全政策

## 支持的版本

| 版本 | 支持状态 |
|------|---------|
| 6.1.x | ✅ 活跃支持 |
| 6.0.x | ✅ 安全更新 |
| < 6.0 | ❌ 不再支持 |

## 报告漏洞

**请勿** 通过公开 GitHub Issue 报告安全漏洞。

请发送邮件至：**security@vibecoding.dev**

邮件应包含：
- 漏洞描述
- 复现步骤
- 影响范围
- 概念验证代码（PoC）
- 您的联系方式

## 响应时间

- **确认**：48 小时内
- **初步评估**：7 天内
- **修复发布**：根据严重程度 1~4 周

## 漏洞奖励

目前暂无正式漏洞奖励计划，但我们会在修复后致谢报告者。
