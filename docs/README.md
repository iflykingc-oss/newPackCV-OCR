## 核心能力

- **多场景提取**: 身份证/营业执照/合同/发票/简历/银行卡...
- **多模型路由**: 豆包/DeepSeek/Kimi/通义千问，自动降级
- **多租户隔离**: 6 层隔离 + 5 级套餐
- **完整计费**: 4 种计费模式 + 月度账单
- **WebHook 生态**: 12 种事件 + 指数退避重试 + DLQ
- **MCP 集成**: AI Agent 直接调用
- **监控告警**: 24 个 Prometheus 指标 + 12 条告警规则
